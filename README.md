
# 24

a game to get 24 from 4 numbers

## Usage

```
runhaskell Main.hs

x1=2
x2=3
x3=8
x4=9
(8.0 / 2.0) * (9.0 - 3.0)
8.0 / (2.0 / (9.0 - 3.0))
8.0 * (9.0 - (2.0 * 3.0))
8.0 * (9.0 - (3.0 * 2.0))
8.0 * ((9.0 - 3.0) / 2.0)
(8.0 * (9.0 - 3.0)) / 2.0
(9.0 - (2.0 * 3.0)) * 8.0
(9.0 - (3.0 * 2.0)) * 8.0
((9.0 - 3.0) / 2.0) * 8.0
(9.0 - 3.0) / (2.0 / 8.0)
((9.0 - 3.0) * 8.0) / 2.0
(9.0 - 3.0) * (8.0 / 2.0)

```
