
# 24

a game to get 24 from 4 numbers

## Usage

```
runhaskell Main.hs

x1=2
x2=3
x3=8
x4=9
(8 / 2) * (9 - 3)
8 / (2 / (9 - 3))
8 * (9 - (2 * 3))
8 * (9 - (3 * 2))
8 * ((9 - 3) / 2)
(8 * (9 - 3)) / 2
(9 - (2 * 3)) * 8
(9 - (3 * 2)) * 8
((9 - 3) / 2) * 8
(9 - 3) / (2 / 8)
((9 - 3) * 8) / 2
(9 - 3) * (8 / 2)
```
